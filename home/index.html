Q&A

Apa itu I Reload Cell?
I Reload Cell adalah platform penukaran point eksklusif bagi pembeli di toko I Reload Cell. Kami memungkinkan Anda untuk menukarkan poin yang Anda peroleh dari setiap pembelian di toko kami untuk berbagai macam hadiah menarik.
Bagaimana cara mendapatkan poin di I Reload Cell?
Pelanggan mendapatkan poin setiap kali mereka membeli produk di toko I Reload Cell. Setiap pembelian Anda akan diberikan code claim untuk mendapatkan point.
Bagaimana cara memasukkan code claim untuk mendapatkan point?

Anda dapat memasukkan code claim dengan mengikuti langkah-langkah berikut:

  1. Login ke akun I Reload Cell Anda.
  2. Klik menu "Dashboard".
  3. Klik tombol "Claim Point".
  4. Masukkan code claim Anda di kolom yang tersedia.
  5. Klik tombol "Klaim Point".
Bagaimana cara menukarkan poin untuk mendapatkan hadiah?
Anda dapat menukarkan poin dengan login ke akun Anda di I Reload Cell dan mengunjungi halaman "Penukaran Point". Di sana, Anda dapat memilih hadiah yang Anda inginkan dan menukarkan poin Anda.
